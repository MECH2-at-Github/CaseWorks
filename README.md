# Installation

Tip: Right click links and then click "Open link in new tab". Tabs can be closed after doing the install.

1. Install a UserScript extension, such as [TamperMonkey (Chrome)](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo). Alternative (untested) extensions are ViolentMonkey and GreaseMonkey.
2. Open [CaseWonks install](https://github.com/MECH2-at-Github/CaseWorks/raw/refs/heads/main/CaseWonks.user.js) and click the [ Install ] button.  
2a. Due to Chrome/Edge browser changes, you must enable "Developer Mode" in Chrome/Edge's Extension's settings area.
   * Chrome: In [Extension settings](chrome://extensions), the Developer Mode slider is located in the top right.
   * Edge: In [Extension settings](edge://extensions/), the Developer Mode slider is located on the left bar.
   * Chrome and Edge will also require a setting to be set in Extensions settings > Tampermonkey. On the Extension settings page, click "Details" under Tampermonkey, and then click the checkbox or slider for "Allow User Scripts".  
    (Chrome visual instructions)
     <img width="800" alt="TM Post-Install-Chrome" src="https://github.com/user-attachments/assets/0af57a7d-dda2-49aa-938e-29c7ba7113de" />
3. Install [Stylus](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
4. Open the [style install page](https://userstyles.world/style/27371/default-slug) and click the [ Install ] button to the bottom right of the image.  
On the next page, click the [ Install Style ] button in the upper left. <img width="600" alt="mec2stylus install" src="https://github.com/user-attachments/assets/ed4dd0cb-e03d-4315-8b43-5b5fc81ef9fc" />

Done!
