# Installation

Tip: Right click links and then click "Open link in new tab". Tabs can be closed after doing the install.

1. Install a UserScript extension, such as [TamperMonkey (Edge)](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd). Alternative (untested) extensions are ViolentMonkey and GreaseMonkey.
1a. Due to Edge browser changes, you must enable "Developer Mode" in Edge's Extension's settings area.
   * Edge: In [Extension settings](edge://extensions/), the Developer Mode slider is located on the left bar.
   * Edge also requires a setting to be set in Extensions > Manage Extensions > Tampermonkey. On the Extension settings page, click "Details" under Tampermonkey, and then click the checkbox or slider for "Allow User Scripts".  
    (Edge visual instructions)
     <img width="800" alt="Edge TM Post-Install" src="https://github.com/user-attachments/assets/ff1e1f10-4ee8-42dc-8f83-a451f3c56586" />

2. Open [CaseWonks install](https://github.com/MECH2-at-Github/CaseWorks/raw/refs/heads/main/CaseWonks.user.js) and click the [ Install ] button.  
3. Install [Stylus](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
4. Open the [style install page](https://userstyles.world/style/27371/default-slug) and click the [ Install ] button to the bottom right of the image.  
On the next page, click the [ Install Style ] button in the upper left. <img width="600" alt="style sheet install" src="https://github.com/user-attachments/assets/ed4dd0cb-e03d-4315-8b43-5b5fc81ef9fc" />

Done!
